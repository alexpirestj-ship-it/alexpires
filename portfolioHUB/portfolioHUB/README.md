# portifolioHUB – Alex Pires

Portfólio pessoal desenvolvido em **HTML5**, **CSS3** e **JavaScript** puro.

## 📁 Estrutura do Projeto

```
portfolioHUB/
├── index.html          # Página inicial (Início)
├── curriculo.html      # Currículo
├── experiencia.html    # Experiência Acadêmica
├── projetos.html       # Projetos
├── css/
│   ├── style.css       # Estilos globais
│   └── inner.css       # Estilos das páginas internas
├── js/
│   └── main.js         # Interações e comportamentos da UI
└── README.md
```

## 🚀 Como usar

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/portfolioHUB.git
   ```
2. Abra `index.html` em qualquer navegador moderno — sem necessidade de servidor.

## 🛠️ Tecnologias

| Camada       | Tecnologia                      |
|--------------|---------------------------------|
| Marcação     | HTML5 semântico                 |
| Estilização  | CSS3 (Custom Properties, Grid, Flexbox) |
| Comportamento | JavaScript (ES5 compatível, sem dependências) |
| Tipografia   | Google Fonts (Playfair Display, Open Sans, Lato, Roboto) |

## ✨ Funcionalidades

- **Responsivo** — Mobile-first, funciona em qualquer tamanho de tela.
- **Acessível** — Skip link, atributos ARIA, navegação por teclado.
- **Menu hamburger** para dispositivos móveis.
- **Reveal on scroll** via IntersectionObserver.
- **Sombra no header** ao rolar a página.
- Embed de apresentação Google Slides.
